Goal: classify canonical Fano 3-folds under some invariant:

\begin{itemize}
  \item Picard number \( \rho(X) \)
  \item Fano index
  \item degree $ c_{1}(X)^{3}=(-K_{X})^{3} $
\end{itemize}

About picard group $ \operatorname{Pic}(X) $ Finitely generated torison-free. and $/ \sim_{\mathbb{Q} } $ coincides with $ /\equiv $, thus
\[
  \mathbb{Z}^{\rho{X}} \cong \operatorname{Cl}(X)/\sim_{\mathbb{Q} } \cong \operatorname{Cl}(X)/\equiv
\]

Weil Fano index $ q_{W}= \max \{ -K_{X}\sim qA, A\in \operatorname{Cl}(X) \} $, and $\mathbb{Q} $-index $ q_{\mathbb{Q} }= \max \{ -K_{X}\sim_{\mathbb{Q} } qA, A\in \operatorname{Cl}(X) \} $
Then
\begin{itemize}
  \item $ q_{W},q_{\mathbb{Q} } \in \mathbb{Z}  $
  \item $ q_{W} | q_{\mathbb{Q} } $
  \item \(X\) smooth, then $ q_{W}=q_{\mathbb{Q} }=i(X) =\max \{ -k_{X}\sim_{\mathbb{Q} }qH, H \in \operatorname{Pic}(X)\}$
\end{itemize}
